# BiteCircuit - Архитектура проекта

## Обзор

BiteCircuit — это комплексная платформа автоматизации для ресторанов и кафе, построенная на современном технологическом стеке с использованием Next.js и Supabase.

## Технологический стек

### Frontend
- **Next.js 14+** - React фреймворк с App Router
- **TypeScript** - типизация
- **Tailwind CSS** - стилизация
- **shadcn/ui** - UI компоненты
- **NextAuth.js** - аутентификация

### Backend
- **Supabase** - Backend-as-a-Service
  - PostgreSQL база данных
  - Realtime subscriptions
  - Storage
  - Edge Functions
  - Row Level Security

### Хостинг
- **Vercel** - деплой Next.js приложения

## Структура базы данных

### Основные таблицы

#### Рестораны и заведения
- `restaurants` - информация о заведениях
- `tables` - столы в заведениях
- `qr_codes` - QR-коды для столов

#### Меню
- `menu_categories` - категории меню
- `menu_items` - позиции меню
- `menu_item_images` - изображения блюд

#### Заказы
- `orders` - заказы
- `order_items` - позиции в заказе
- `order_status_history` - история статусов

#### Пользователи и аутентификация
- `users` - пользователи (клиенты)
- `restaurant_staff` - персонал заведений
- `roles` - роли пользователей

#### Лояльность
- `loyalty_programs` - программы лояльности
- `user_points` - баллы пользователей
- `rewards` - награды и бонусы

#### Бронирование
- `reservations` - бронирования столов
- `waitlist` - очередь ожидания

## Безопасность

### Row Level Security (RLS)
- Пользователи видят только свои заказы
- Персонал видит только заказы своего заведения
- Администраторы имеют полный доступ

### Аутентификация
- NextAuth.js с Supabase адаптером
- OAuth провайдеры (Google, Telegram)
- Magic links

## Масштабирование

- Vercel Edge Functions для быстрого отклика
- Supabase Realtime для live обновлений
- CDN для статических ресурсов
- Кэширование на уровне Next.js

## Мониторинг и аналитика

- Error tracking (будет определено)
