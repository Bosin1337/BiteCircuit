# BiteCircuit - Product Requirements Document

## 1. Базовые must-have функции

### QR-меню + заказ со стола
- **Описание**: клиент сканирует QR → смотрит меню → заказывает → оплачивает (нужна система эквайринга)
- **Преимущества**: быстрее обслуживание, меньше ошибок, выше средний чек
- **Приоритет**: ✅ обязательно

### Онлайн-заказ (веб / PWA / TMA)
- **Описание**: заказ заранее (takeaway / доставка)
- **Преимущества**: рост выручки, удобство, меньше очередей
- **Приоритет**: ✅ обязательно

### Оплата без кассы (QR / Apple Pay / Google Pay)
- **Описание**: оплата прямо со смартфона
- **Преимущества**: ускорение оборота столов
- **Приоритет**: ✅ обязательно

## 🤖 2. Автоматизация взаимодействия

### Чат-бот (Telegram / WhatsApp / Web)
- **Описание**: меню, заказ, акции, FAQ
- **Преимущества**: 24/7 обслуживание, снижение нагрузки
- **Приоритет**: ✅ да

### Голосовой бот (заказ по телефону)
- **Описание**: автоответчик принимает заказы
- **Преимущества**: экономия на операторе
- **Приоритет**: ⚖️ опционально

### Push / уведомления (через web app или Telegram)
- **Описание**: акции, напоминания
- **Преимущества**: возврат клиентов
- **Приоритет**: ✅ да

## 🎯 3. Персонализация и лояльность

### Программа лояльности (баллы / кэшбэк)
- **Описание**: бонусы за покупки
- **Преимущества**: рост повторных визитов
- **Приоритет**: ✅ обязательно

### AI-рекомендации блюд
- **Описание**: "вам понравится это"
- **Преимущества**: увеличение среднего чека (официант)
- **Приоритет**: ⚖️ желательно

### Персональные предложения (CRM)
- **Описание**: акции под конкретного клиента

- **Приоритет**: ✅ да

## 🧾 4. Удобство посещения

### Очередь / waitlist онлайн
- **Описание**: "ваш стол через 10 мин"
- **Преимущества**: снижение негатива
- **Приоритет**: ⚖️ если поток большой

### Статус заказа (как в доставке)
- **Описание**: "готовится / готово"
- **Преимущества**: прозрачность
- **Приоритет**: ✅ да

### Онлайн-бронирование столов
- **Описание**: через сайт / бот
- **Преимущества**: меньше хаоса в часы пик
- **Приоритет**: ⚖️ зависит от формата
